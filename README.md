# SolarVerse VR — AI-гид по Солнечной системе

**SolarVerse VR** — VR-симулятор Солнечной системы для **Meta Quest**, где пользователь перемещается между планетами и изучает их в интерактивном режиме. При выборе планеты открывается UI-панель, в которой можно вести диалог с **AI-гидом**: задавать вопросы, получать объяснения и (опционально) запускать мини-задания/квизы.

---

## Возможности (MVP)
- VR-навигация между планетами (перемещение).
- Интерактивные объекты планет: клик по планете → открытие UI-панели.
- AI-диалог с контекстом выбранной планеты (название/параметры/описание).
- Локальная демонстрация.

---

## Технологии
- **Unity 6**
- **XR Interaction Toolkit**
- **OpenXR**
- **Meta Quest**
- **OpenAI API**

---

## Архитектура (кратко)
Поток работы:
1) Пользователь кликает по планете в VR  
2) Приложение формирует контекст (planet_name + параметры)  
3) Запрос отправляется в OpenAI API  
4) Ответ отображается в UI-панели ( и в планах добавить озвучивание)

---

## Требования
- Unity 6 (версия проекта)
- Установленные пакеты: **XR Interaction Toolkit**, **OpenXR**, **Input System**
- Meta Quest (для VR-демо) + включённый **Developer Mode**
- Действующий **OpenAI API Key**


Запуск на Meta Quest (VR Demo)

- В Unity переключите платформу на Android:

    File → Build Settings → Android → Switch Platform

- Включите OpenXR:

    Project Settings → XR Plug-in Management → (Android) → OpenXR

- Подключите Meta Quest по USB (или через Wi-Fi ADB), убедитесь что включён Developer Mode.

- Нажмите Build And Run.

- В шлеме:

    найдите/выберите планету (лучом/кликом),

    откройте UI-панель,


    задайте вопрос → получите ответ AI-гидa.
